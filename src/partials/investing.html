Ежегодный обратный выкуп будет рассчитываться по следующей формуле: «Объем проданных токенов NODE * 2 / Миллион»

То есть в случае достижения уровня продаж в 10 млн токенов NODE, у всех инвесторов, у которых после обратного выкупа на балансе будут токены NODE, получат дополнительно 20%.

Таким образом, инвестор может уже сейчас рассчитать гарантированную процентную прибыль, которую он будет получать ежегодно от выкупа токенов, вне зависимости от спроса токенов NODE на рынке. Соответственно, чем дольше инвестор держит токены, тем больше дохода он сможет получить.

Токены NODE, которые компания приобретет в результате обратного выкупа, будут распределены между существующими инвесторами пропорционально текущему количеству токенов на балансе на момент такого распределения.

В любой момент времени у покупателя будет возможность приобрести продукцию NODE с 25% скидкой по отношению к стоимости в фиатной валюте (USD) при условии оплаты токенами компании. Например, при фактической стоимости Power pad в 99,0USD, цена покупки при оплате токенами NODE составит 74,25 USD. Токены, которые компания приобретет в результате продажи своей продукции, будут сожжены, таким образом уменьшая их общее количество в обороте.

ПРОГНОЗ СПРОСА НА ТОКЕНЫ NODE

Минимум 50% покупок через прямой канал продаж будут произведены токенами. Общая планируемая доля прямых продаж – 20%. Расчетный годовой спрос в итоге составит 10 млн. токенов общей стоимостью 8 млн. USD, без учета спекулятивной составляющей. Это обеспечит минимум 2-х кратную оборачиваемость на вложенный рабочий капитал.
